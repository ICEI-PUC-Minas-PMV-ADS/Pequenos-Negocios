# PREJETO PEQUENOS NEGÓCIOS

`ANÁLISE E DESENVOLVIMENTO DE SISTEMAS`

`DESENVOLVIMENTO DE APLICAÇÃO WEB FRONT-END`

`PRIMEIRO SEMESTRE`

Criação de um site responsivo que forneça todo o conteúdo necessário para o um pequeno negócio, a fim de fomentar a visibilidade dos produtos/serviços e meios de contato.
